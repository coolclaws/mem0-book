# 附录 A　推荐阅读路径

## 按目标选择阅读路径

### 🚀 快速上手（2小时）

目标：能在项目中集成 mem0，理解基本用法

```
第 1 章（概览）→ 第 3 章（配置）→ 第 5 章（add）→ 第 6 章（search）→ 第 20 章（框架集成）
```

### 🔍 深度理解核心管线（半天）

目标：完整理解 mem0 的记忆管线，能定制 Prompt 和排查问题

```
第 1-8 章（全部读完）
重点：第 5 章（add 管线）、第 7 章（更新决策）、第 16 章（Prompt 工程）
```

### 🏗️ 生产部署（半天）

目标：在生产环境中可靠运行 mem0

```
第 3 章（配置）→ 第 9-10 章（向量库选型）→ 第 11 章（Embedder）→ 第 17 章（SDK）→ 第 19 章（REST Server）
```

### 🕸️ 图记忆专项（2小时）

目标：启用并理解图记忆系统

```
第 13 章（MemoryGraph）→ 第 14 章（实体关系）→ 回看第 5 章中的图写入部分
```

### 🔧 二次开发（1天）

目标：为 mem0 新增向量库/LLM 适配器，或做深度定制

```
第 2 章（仓库结构）→ 第 9 章（向量抽象层）→ 第 15 章（LLM 抽象层）→ 阅读现有适配器源码
```

### 🆚 技术选型（1小时）

目标：在 mem0 和其他记忆框架之间做选型

```
第 1 章 1.7-1.9 节（同类项目对比）→ 根据自己的场景查阅对应章节
```

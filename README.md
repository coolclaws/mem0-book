# mem0 源码解析

深入剖析 mem0 —— AI Agent 智能记忆层的设计与实现

📖 在线阅读：https://mem0-book.myhubs.dev

## 关于本书

本书深入解析 [mem0](https://github.com/mem0ai/mem0) 项目的源码，涵盖：
- 核心记忆管线（add/search/update）
- 20+ 向量数据库适配器
- 图记忆系统（Neo4j）
- LLM/Embedder/Reranker 集成层
- Python SDK + TypeScript SDK + REST API Server
- LangGraph / CrewAI / AutoGen 框架集成

